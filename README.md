## ğŸš€ Getting Started

First, clone the repository and install dependencies:

```bash
git clone https://github.com/YogiJi21/my-portfolio-next-app.git
cd my-portfolio-next-app
npm install
```

Then, run the development server:

```bash
npm run dev
# or

# or

```
